Tipos de Etiquetas

En HTML y CSS tenemos diferentes tipos de etiquetas cada uno con un propósito en particular una etiqueta puede componerse de dos maneras.

ESTRUCTURA DE ETIQUETAS
Etiquetas que solo se aperturan Etiquetas que tienen una apertura y un sierre
Este tipo de etiqueta comúnmente se utilizan para indicar una dirección u origen, se componen por argumentos u opciones tales como src, href. Este tipo de etiquetas común mente se utilizan para encontrar un contenido tales como texto, img, entre otros.

Listado de etiquetas dirigidas al texto

  • <p></p> Se utiliza para colocar un texto
  • <h1></h1> Se utiliza para colocar un titulo

Listado de etiquetas dirigidas a objetos

  • <img> Se utiliza para colocar una imagen
  • <header> Se utiliza para colocar un encabezado
  • <section> Se utiliza para dividir en secciones
  • <a> </a> Se utilza para colocar un link

Listado de propiedades css dirigidas a texto

  • color: #ffffff; Se utiliza para darle color al texto
  • font-size: 1px; Se utiliza para cambiar de tamaño el texto
  • font-family: Se utiliza para cambiar la fuente del texto
  • text-align: Se utilza para alinear un texto

Listado de propiedad css dirigidas a objetos

  • width: 100px; Se utiliza para cambiar el ancho de una imagen
  • height: 100px; Se utiliza para cambiar el alto de una imagen
  • border: 30px; Se utiliza para agregar un borde a una imagen o a un texto
